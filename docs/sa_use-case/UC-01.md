# UC-01

## Создание шаблонных сообщений операторами интегрируемых сервисов.

## Акторы: Оператор, системный администратор.

## Триггер: Требуется создать шаблонное сообщение для постоянного использования сервисами.

Основной сценарий:
    1. Пользователь заполнил форму шаблонного сообщения.
    2. Система проверяет корректность данных.
    3. Система сохраняет шаблон сообщения в базу данных.

Альтернативный сценарий:
    п.1
    2. Данные введены не корректно.
